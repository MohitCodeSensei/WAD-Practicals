About this project

Learning responsive web design and modern CSS practices

What is this?

This is a WAD (Web Application Development) practical assignment for first-year students. It's a simple, clean showcase of modern tech companies with responsive design principles applied throughout.

Project goals

  • Build responsive layouts using CSS Grid and Flexbox
  • Implement mobile-first design with media queries
  • Create clean, accessible navigation systems
  • Learn about semantic HTML and ARIA attributes
  • Practice basic JavaScript for interactivity (menu toggle)

Technologies used

  • HTML5 — semantic markup, accessibility features
  • CSS3 — Grid, Flexbox, animations, gradients, media queries
  • JavaScript — menu toggle, lazy-loading images, event handling

Featured companies

The site showcases four visionary companies:

  • SpaceX — Reusable rockets and Mars exploration
  • Tesla — Electric vehicles and sustainable energy
  • OpenAI — Artificial intelligence research and safety
  • NVIDIA — GPU technology and AI computing

Learning outcomes

Through this project, students learn to:

  • Design and build responsive, multi-page websites
  • Use modern CSS techniques (Grid, Flexbox, animations)
  • Create accessible navigation and interactive elements
  • Write clean, maintainable HTML/CSS/JS code
  • Understand mobile-first design philosophy
  • Implement SEO best practices and semantic HTML
